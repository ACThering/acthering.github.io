Images: Alt Text, Long Descriptions, and Decorative Images

Use AI to draft text alternatives quickly, then apply human judgment. Validate before posting to Brightspace.

AI can draft alt text and long descriptions, but it does not determine what is instructionally essential. Provide purpose and context, review carefully, and validate in Brightspace.

Fix or rebuild: choosing the most effective path

For images, accessibility remediation often starts with a decision. In many cases, replacing an image with accessible text, a simple table, or a redesigned visual is more effective than attempting to describe a complex or low-quality image. The goal is not to preserve the original image at all costs but to ensure learners can access the instructional meaning.

  • Fix the image use when it is instructionally meaningful, visually clear, and can be described accurately with alt text or a long description.
  • Rebuild or replace when the image contains dense text, low-resolution content, complex charts without underlying data, or decorative styling that obscures meaning.

Reminder: replacement is an accessibility strategy

Decorative images: choosing the right approach

Not every image needs alt text. Images that do not convey instructional meaning should be marked as decorative so they are skipped by screen readers.

Mark as decorative when the image does not convey instructional meaning. The image can be removed without affecting what students are expected to learn.

Write alt text when the image communicates essential information that can be conveyed accurately in one or two sentences.

Provide a long description when the image is complex and cannot be adequately described in one or two sentences, such as charts, diagrams, or multi-part visuals.

Decorative, alt text, or long description: quick decision guide
If the image Then choose What to do
Does not add instructional meaning and is not referenced in instructions Mark as decorative Check “decorative” so screen readers skip the image. In most learning platforms, this is preferable to leaving alt text blank.
Conveys instructional meaning that can be explained briefly Alt text Write 1 to 2 concise sentences describing the essential information
Is complex or multi-part and cannot be explained briefly Long description Provide alt text plus a longer text description placed near the image

Choosing the simplest option that fully communicates the instructional meaning is the most accessible approach.

Common examples
  • Decorative: Stock photo banner used for visual interest only
  • Decorative: Icon that repeats adjacent text (for example, a calendar icon next to a due date)
  • Alt text: Screenshot that shows where to click to submit an assignment
  • Long description: Multi-part diagram or infographic where relationships and labels matter
Common pitfalls
  • Adding alt text to decorative images, which creates unnecessary repetition and distraction for screen reader users
  • Marking an image as decorative when students are expected to interpret it
  • Using long descriptions for simple images that can be described in one sentence

Alt text for images

Typical uses include photos, diagrams, screenshots, icons that convey meaning, and instructional images embedded in Brightspace pages or documents.

How to upload and use the prompts (Alt text)
  1. Use an institutionally approved AI tool when working with university or student-related content.
  2. Prepare the image and remove student identifiers. If the image includes student names, grades, or personal information, do not upload it.
  3. Provide the image in one of these ways:
    • Upload the image file if file upload is available.
    • If upload is not available, paste a description of the image and any visible text.
  4. Provide one sentence of context that explains the purpose of including the image in the course and who the audience is.
  5. Paste one prompt from the prompt bank below.
  6. Review the result and revise as needed to match instructional intent and course terminology.
  7. Enter the final alt text in the tool where the image is posted. Validate in Brightspace.
Prompt bank (Alt text)

Label: purpose based alt text

Prompt A. Purpose based alt text draft

You may be receiving an uploaded image, or you may only receive a pasted description of the image.

First, ask me to provide:
1) The instructional purpose of including this image in the course
2) What the image is meant to teach or demonstrate
3) Who the intended audience is
4) Any text that appears in the image that must be conveyed

If you have enough information, draft concise, purpose based alt text in 1 to 2 sentences.
Include only essential information needed to achieve the instructional purpose.
Avoid phrases such as "image of" or "picture of" unless needed for clarity.
If the image appears decorative, recommend marking it as decorative instead of writing alt text.

Return the alt text only.

Prompt B. Decorative or informative decision support

You may be receiving an uploaded image, or you may only receive a pasted description of the image.

Based on the image and the course context I provide, determine whether the image is:
1) Informative and requires alt text
2) Decorative and should be marked as decorative

If informative, draft 1 to 2 sentences of alt text.
If decorative, state "Mark as decorative" and provide one short rationale.

Return the recommendation and the alt text if applicable.
Manual remediation checklist for Alt text
  • Confirm whether the image is decorative or informative using the decision guide above.
  • For informative images, write alt text that communicates the instructional purpose, not just visual appearance.
  • Keep alt text concise and focused on essential meaning.
  • Include visible text only when it is instructionally necessary.
  • Avoid duplicating surrounding page text in the alt text.
  • For complex visuals, use a long description in addition to alt text.
Validation for Alt text
  • Confirm the image has alt text or is marked as decorative in the authoring tool.
  • Run the Ally Course Report for Brightspace pages that include images.
  • Spot check with a screen reader to confirm the alt text is read appropriately and in context.

Typical time: 2 to 10 minutes per image

Common pitfalls in Alt text
  • Alt text describes appearance but not instructional meaning.
  • Alt text is too long and includes unnecessary detail.
  • Decorative images have alt text, creating noise for screen reader users.
  • Images contain text that should be replaced with real text on the page when possible.

Long descriptions for complex images

Use long descriptions for charts, graphs, infographics, process diagrams, maps, and complex visuals where 1 to 2 sentences cannot convey the instructional meaning.

How to upload and use the prompts (Long descriptions)
  1. Use an institutionally approved AI tool and remove student identifiers.
  2. Provide the image by uploading it or by pasting a detailed description of what it contains, including any labels and visible text.
  3. Provide context that explains the instructional purpose of the image and what students should learn from it.
  4. If the image is a chart or graph, provide the underlying data if available. If data is not available, provide the best available description of axes, legend, and key values.
  5. Paste one prompt from the prompt bank below.
  6. Review the output for accuracy and revise as needed.
  7. Post the long description near the image in Brightspace, using clear labeling such as "Long description" or "Text alternative."
Prompt bank (Long descriptions)

Label: text alternatives for complex visuals

Prompt A. Long description draft

You may be receiving an uploaded complex image, or you may only receive a pasted description of the image.

First, ask me to provide:
1) The instructional purpose of this image
2) What it is meant to teach or demonstrate
3) Who the intended audience is
4) Any visible text, labels, axes, or legend items that must be conveyed
5) If it is a chart, any data values available

Then write:
1) Alt text in 1 to 2 sentences that identifies the image type and main point
2) A long description that stands alone and communicates the essential information

Long description requirements:
- Begin with a 1 to 2 sentence overview
- Follow with an objective description of 4 to 8 sentences
- Describe structure, relationships, sequence, and key values or trends
- Include readable versions of any text in the image that is required for meaning
- Avoid interpreting conclusions that are not directly supported by the image

Return the alt text and long description only.

Prompt B. Chart or graph text alternative with data table recommendation

You may be receiving an uploaded chart or graph, or you may only receive a pasted description.

If the underlying data is available, recommend whether a simple data table should be posted in Brightspace as a companion text alternative.
If no data is available, recommend what data would be needed to create a table.

Then draft:
1) Alt text in 1 to 2 sentences
2) A long description in 4 to 8 sentences focused on key trends and values

Return the recommendation, alt text, and long description.
Manual remediation checklist for Long descriptions
  • Confirm whether a long description is needed. If the image is complex, 1 to 2 sentences are not sufficient.
  • Ensure the long description communicates structure and relationships, not decorative styling.
  • For charts and graphs, consider providing a data table or a short summary of key values.
  • Place the long description near the image and label it clearly.
  • Ensure the long description does not repeat surrounding text verbatim unless needed for meaning.
Validation for Long descriptions
  • Confirm the image has alt text and that the long description is discoverable near the image.
  • Run the Ally Course Report for the page where the image is posted.
  • Spot check with a screen reader to confirm that users can reach the long description easily.

Typical time: 10 to 30 minutes per complex image

Common pitfalls in Long descriptions
  • Long descriptions are missing labels, axes, or legend information.
  • Descriptions include interpretation not supported by the image.
  • Descriptions are posted far from the image or without a clear label.
  • Charts are presented without underlying data or a usable text alternative.

Posting images in Brightspace

Where you place text alternatives matters. Use consistent labeling and ensure students can find the information without searching across pages.

Recommended placement patterns
  • Alt text belongs in the image properties field in Brightspace or in the source file where the image is embedded.
  • Long descriptions should be placed immediately below the image or linked directly with clear link text such as "Long description of chart."
  • If the image is central to an assignment or assessment, include the text alternative in the same content item as the image.
  • If the image contains essential text, replace the image with real text on the page when possible.
Quick prompt to rewrite image text as real text 
You may be receiving an uploaded image that contains text, or you may receive pasted text from the image.

If text content is present, rewrite it as clean, accessible plain text suitable for posting directly on a Brightspace page.
Preserve headings and list structure.
Do not add content that is not present.

If no readable text is present, briefly state what content is needed to proceed.

Return plain text only.
